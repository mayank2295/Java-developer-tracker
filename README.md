# Java Developer Tracker

A structured roadmap and learning tracker for becoming a Java Full Stack Developer.

It covers Java, Spring Boot, React, JWT authentication, Docker, Kafka, Azure/AWS, backend architecture, real-world projects, and industry-focused development preparation.

## Open The Platform

Open this file in your browser:

```text
java-fullstack-platform/index.html
```

The app currently runs as a static frontend and saves progress, notes, and custom tasks in browser `localStorage`.
